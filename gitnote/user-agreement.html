二、服务内容

1. Git记 是一款本地优先的 Markdown 笔记应用，主要用于创建、编辑、管理、预览和同步个人笔记。

2. Git记 的核心功能可能包括：

  • 创建和管理本地 Git 笔记仓库；
  • 通过 HTTPS 或 SSH 方式连接你自行选择的 Git 仓库；
  • 创建、编辑、移动、删除、收藏和搜索 Markdown 笔记；
  • 管理文件夹、图片附件、最近修改、最近删除和冲突文件；
  • Markdown 预览、表格预览、Mermaid 图表预览和编辑器快捷工具；
  • 一键同步、提交、拉取、合并、推送及冲突处理；
  • App 密码、系统生物识别解锁等本地安全能力；
  • 免费版和专业版相关功能权益。

3. Git记 以本地处理为主。你的笔记通常以 Markdown 文件形式保存在设备本地或你自行配置的 Git 仓库中。你理解并同意，Git记 不是云笔记托管平台，我们通常不为你托管、备份或恢复笔记内容。
